## ğŸ› ï¸ INSTALLATION & SETUP

Since this is a static site, you don't need a backend server.

### 1. Clone or Download
```bash
git clone https://github.com/aaron-devop/darkcore-landing.git
```

### 2. Edit Content
Open `index.html` in any code editor (VS Code, Notepad++, etc.) and look for these sections to customize:

* **Discord Link:** Search for `https://discord.gg/your-code` and replace it.
* **Server IP:** Search for `play.darkcore.hu`.
* **Player Count:** Search for `124`.

### 3. Deploy
Upload the `index.html` to **GitHub Pages**, Netlify, or your own VPS web root (e.g., `/var/www/html`).

You can easily change the color scheme by modifying the CSS Variables at the top of the `<style>` section:

```css
    --accent-secondary: #ec4899; /* Secondary Neon Color (Pink) */
}
```
